# EduGestión - SaaS para Gestión de Academias

Sistema de gestión integral para academias y tutores independientes.

## Stack Tecnológico
- Angular 17+ (Standalone Components, Signals)
- TailwindCSS
- Supabase (Auth, Database, Storage)
- Stripe (Pagos)

## Estructura del Proyecto
```
src/
├── app/
│   ├── core/
│   ├── shared/
│   ├── features/
│   └── layouts/
```

## Instalación
```bash
npm install
ng serve
```
